This project helps you download YouTube videos and audio using Python.
It is built using the pytube library, which makes downloading YouTube content very easy.

### Requirements: Before running this project, make sure Python is installed on your system and you have an active internet connection.

### Installation: You need to install the pytube library. Open your terminal / command prompt and run
                        pip install pytube
                        from pytube import YouTube
            
## How This Project Works
    1. The user provides a YouTube video link
    2. The program connects to YouTube using pytube
## You can:
    Download the video
    Download audio only
    The file is saved on your local system
    
## Features
    Download YouTube videos
    Download audio from YouTube videos
    Simple and easy to understand code
    Beginner-friendly project
